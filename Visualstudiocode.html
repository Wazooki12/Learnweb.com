Wie installiere ich Visual Studio Code?

  1. Schritt 1: Besuche die Website
    Gehe auf https://code.visualstudio.com und lade Visual Studio Code herunter.
  2. Schritt 2: Wähle dein Betriebssystem
    Klicke auf den Download-Button für dein Betriebssystem:
    • Windows: „Download for Windows“
    • macOS: „Download for macOS“
    • Linux: Wähle die Version für Linux.
  3. Schritt 3: Installieren
    Öffne die heruntergeladene Datei und folge den Installationsanweisungen.
  4. Schritt 4: Starte Visual Studio Code
    Öffne das Programm nach der Installation:
    • Windows: Im Startmenü
    • macOS: Im „Programme“ Ordner
  5. Schritt 5: Installiere Erweiterungen (optional)
    Installiere die „Live Server“ Erweiterung, um deine Webseite direkt im Browser zu sehen.
  6. Schritt 6: Starte dein erstes Projekt
    Jetzt kannst du deine ersten HTML und CSS Projekte in Visual Studio Code erstellen!
