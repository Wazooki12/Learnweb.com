Über dieses Projekt

Willkommen bei meiner Website LearnWeb! Das ist mein Schulprojekt für 2025, bei dem ich eine Website erstellt habe, die dir zeigt, wie man die Grundlagen einer Webseite in Visual Studio Code lernt. Mein Ziel war es, mir selbst beizubringen, wie man eine Website erstellt, damit ich später anderen zeigen kann, wie sie das Gleiche tun können.

Auf LearnWeb findest du einfache Anleitungen und Tutorials, die dir Schritt für Schritt die Grundlagen von HTML und CSS beibringen. Du lernst auch, wie du Visual Studio Code dafür richtig nutzt.

Mir war es auch wichtig, dass ich beim Projekt Spass habe und nicht nur trockene Theorie lerne. Ich hoffe, du hast genauso viel Spass beim Lernen und Entdecken der Grundlagen, um deine eigene Webseite zu erstellen!
