Was ist CSS?

CSS (Cascading Style Sheets) wird verwendet, um das Aussehen und Layout von Webseiten zu gestalten.

Grundlegende CSS-Eigenschaften:

  • color: Bestimmt die Textfarbe.
  • font-family: Legt die Schriftart fest.
  • background-color: Setzt die Hintergrundfarbe.
  • margin: Bestimmt den Aussenabstand eines Elements.
  • padding: Bestimmt den Innenabstand eines Elements.

Beispiel:

            body {
                background-color: #f0fdf4;
                font-family: Arial, sans-serif;
            }
            h1 {
                color: #047857;
            }
