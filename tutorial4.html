Was sind Formulare?

Formulare ermöglichen es den Benutzern, Daten auf einer Webseite einzugeben und zu übermitteln. Sie sind in HTML mit dem <form>-Tag erstellt.

Grundlegendes Beispiel für ein Formular:

            <form action="submit.php" method="post">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name">
                <input type="submit" value="Absenden">
            </form>

Wichtige Formular-Elemente:

  • <input>: Zur Eingabe von Text, Zahlen, oder zum Erstellen von Schaltflächen.
  • <textarea>: Zum Eingeben mehrzeiliger Texte.
  • <select>: Zum Erstellen von Dropdown-Menüs.
  • <button>: Zum Erstellen von Buttons innerhalb eines Formulars.
