Übung 1: HTML Grundstruktur

Erstelle in Visual Studio Code eine HTML-Datei mit folgendem Inhalt:

  • Ein <h1>-Tag mit dem Text „Willkommen auf meiner Webseite“
  • Ein <p>-Tag mit einer kurzen Begrüßung

Übung 2: CSS-Grundlagen

Gestalte die Hintergrundfarbe der Webseite mit CSS in Visual Studio Code und setze die Textfarbe des <h1>-Tags auf „#047857“.

Übung 3: Links und Bilder

Erstelle eine Webseite in Visual Studio Code, die:

  • Einen Link zu einer externen Webseite (z. B. Wikipedia) enthält
  • Ein Bild deiner Wahl anzeigt und es mit einem passenden alt-Text beschreibt

Übung 4: Formulare

Erstelle ein Formular in Visual Studio Code, das folgendes beinhaltet:

  • Ein Textfeld zur Eingabe des Namens
  • Ein Absende-Button, der die Formulardaten sendet

Übung 5: Tabellen

Erstelle in Visual Studio Code eine Tabelle, die die Namen und das Alter von 3 fiktiven Personen anzeigt.

Übung 6: SEO-Grundlagen

Füge in Visual Studio Code einen Meta-Titel und eine Meta-Beschreibung für deine Webseite hinzu. Achte darauf, dass du relevante Keywords einbaust.
